Uma simples lista de tarefas com `React` que roda apenas no navegador.

## ▶️ Como executar

> [!TIP]
> Para executar o projeto é preciso ter o `Node` instalado.

1. **Clone e acesse o repositório**

    ```git
    git clone https://github.com/Davi-1903/ToDoList.git
    cd ToDoList
    ```

2. **Instale todas as dependências**

    ```bash
    npm install
    ```

3. **Execute a aplicação**

    ```bash
    npm run dev
    ```

Após a execução do passo a passo, a aplicação iniciará no seu navegador padrão na porta [`http://localhost:3000`](http://localhost:3000)
